My Projects

In-Focus Mode

The "In Focus Mode" project is a digital productivity and concentration enhancement tool designed to help individuals improve their focus and increase their productivity. In today's fast-paced world, distractions are abundant, and maintaining concentration on tasks can be challenging. This project aims to provide a solution that assists users in staying on track and achieving their goals more efficiently.

Home Security System

The "Home Security System Summary Project" is an initiative aimed at designing and implementing an advanced, integrated home security system. With the rising concerns about safety and security, this project addresses the need for a comprehensive solution that not only safeguards homes against various threats but also provides real-time monitoring and control for homeowners.

Driver Drowsiness System

A important project called "Driver Drowsiness Detection System" aims to improve road safety by reducing crashes brought on by sleepy or tired drivers. Advanced technology is used by this system to monitor the driver's condition and to automatically stop the car if indicators of lack of attention or sleepiness are found.
